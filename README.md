# Customer Churn Prediction

## Project Overview
Predict customer churn using machine learning.

## Dataset
Telco Customer Churn dataset.

## Goal
Identify customers likely to churn and provide business insights.

## Methods
- Exploratory Data Analysis
- Feature Engineering
- Logistic Regression
- Random Forest
- XGBoost

## Results
Best model: XGBoost
ROC-AUC: 0.84
